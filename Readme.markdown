Ejercicio de Angular.js
=======================


Para correr clonar repo:

    $ git clone git@kujenga.co:macario/opi-angular-exercise.git accounts
    $ cd accounts
    $ npm install


Para correr el servidor de desarrollo:

    $ gulp


Visitar http://localhost:8080
